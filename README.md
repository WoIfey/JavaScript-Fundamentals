## Vad är JavaScript?

JavaScript förkortas ofta som JS, men när vi talar om språket brukar vi använda hela namnet. Nyckeln till vad det är för typ av språk finns i namnet: JavaScript är ett skriptspråk. Till skillnad från HTML och CSS är JavaScript som ett programmeringsspråk, eftersom det kan utföra riktiga kommandon.
